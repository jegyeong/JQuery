** html()메소드와 text()메소드
지정된 엘리먼트(선택자)를 대상으로 값을 가져온다 - getter의 의미
** html(값)메소드와 text(값)메소드
지정된 엘리먼트(선택자)를 대상으로 값을 출력,변경한다 - setter의 의미


htmt :태그를 포함한 문장을 가져온다
text :태그를 포함하지 않는 문자만 가져온다

*선택자가body인경우
$("body").html() // body 태그 안의 모든 태그를 포함한 문장
$("body").text() // body 태그안의 모든 문자만

*선택자가 body가 아닌 경우
$("p").text() //p가 여러개인경우 모든 p태그의 문자만
$("p").html() //p가 여러개인경우 첫번째거만 태그를 포함한 문장
* html()메소드 사용시 첫번째거만 대상으로 가져온다
전체를 대상으로 가져오기 위해서는 반복루프를 사용한다
$.each($('p'), function() { } )
$('p').each(function(){ })
