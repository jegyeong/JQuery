수행내용

get(0): jQuery객체로부터 지정(0)번째 객체를 얻는다
get()메소드가 리턴하는 객체는 dom객체이다
dom객체는 jQuery메소드를 사용 할 수 없다

dom객체가 jQuery메소드를 사용하기 위해서는
스크립트 문법으로 수행 하던지 또는
dom객체를 jQuery 의 $()로 감싸주면 가능하다

var jq= $('h1');
var dom = jq.get(0); // dom객체를 리턴
dom.style.color = 'red';
$(dom).css('color', 'blue');
